# ML Models

This folder contains the machine learning models and datasets used for the Tarass project.

## Models
- `crowd_forecast_model.ipynb` : Predicts future crowd levels at service centers
- `no_show_prediction_model.ipynb` : Predicts the likelihood of no-shows

## Datasets
- `data/crowd_data.csv` : 40,000 records of historical crowd levels
- `data/no_show_data.csv` : 40,000 records of user no-shows

## How to Use
1. Load datasets in Python using pandas.
2. Install required libraries: `pip install -r ../requirements.txt`
3. Run the notebooks in Google Colab or Jupyter to train/test models.
